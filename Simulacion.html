conceptos



¿que es un sistema de fuente?

el sistema de fuentes es el entorno real o virtual que estamos interesados en moderar, en la cual constitutye una fuente de datos observables en la forma de trayectoria

¿base de datos del comportamiento?

son observaciones o caracterisiticas de los datos que son observados o adquiridos a travez de marcos experimentales de interes para quien realiza el modelo y pueda tener una efectiva solucíon

¿marco experimental?

es el que establece un conjunto de experimentos para los cuales sera muy útil para el análisis, el diseño y la optimización

Validez

La validez representa la relacion del sistema y el modelo dentro del marco experimental a tal grado que nos permite dar con certeza la fiabilidad del proceso que lleva el modelo dentro del sistema real del mundo dentro de un rango satisfactio de consistencia.

Validez replicativa

Cuando se habla de la validez replicativa se denomina como la posibilidad dentro de cada experimento llevado acabo en el modelo teniendo en cuenta su relacion con el sistema real del mundo dado asi multiples visiones de validez dentro del modelo.

Validez predictiva

Dentro de la validez encontramos la necesidad de usar la prediccion para poder denotar o identificar posibles sucesos dentro del modelo que se pueden reflejar en un futuro dentro del sistema real del mundo, dando asi solucion a posibles sucesos futuros viendolos dentro del modelo realizado.

modelo de sistema

Conjunto de instrucciones, ecuaciones, reglas y ligaduras que reproducen el comportamiento del sistema. Especificación del conocimiento acerca del comportamiento del sistema a: El nivel 2 (Generación) El nivel 3 (Estructura) Necesariamente dispone de un agente capaz de obedecer estas instrucciones y reglas.
