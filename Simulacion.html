conceptos



¿que es un sistema de fuente?

el sistema de fuentes es el entorno real o virtual que estamos interesados en moderar, en la cual constitutye una fuente de datos observables en la forma de trayectoria

¿base de datos del comportamiento?

son observaciones o caracterisiticas de los datos que son observados o adquiridos a travez de marcos experimentales de interes para quien realiza el modelo y pueda tener una efectiva solucíon

¿marco experimental?

es el que establece un conjunto de experimentos para los cuales sera muy útil para el análisis, el diseño y la optimización

Validez

La validez representa la relacion del sistema y el modelo dentro del marco experimental a tal grado que nos permite dar con certeza la fiabilidad del proceso que lleva el modelo dentro del sistema real del mundo dentro de un rango satisfactio de consistencia.
